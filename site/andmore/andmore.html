架空サイト
「北海道札幌飛世高等学校」

コンセプト
「グローバル」と「多様性」が学べる公立高校。

ターゲット
中学生とその保護者

見やすいサイトにすることを心がけました。複数ページにすることで、1ページあたりのスクロール量を抑えました。中学生が見そうな「学校の特色」部分は明るめの配色に、保護者が見そうな「学校の概要」は落ち着いた配色にしました。

様々な機能を使うことで、より使いやすく見やすくなると思い、適宜使用しています。（slick slider / アコーディオン / 左側にナビ固定 / ナビ内の階層 / フェードアップ / ファビコン）

職業訓練校の制作課題で作成しました。学校の内容は問わなかったので、自分が行っていた高校をイメージしながら制作しました。

制作時間
デザイン＋コーディングで9日間

制作時期
2023年8〜9月頃

レスポンシブ対応済み。

使用ツール:Visual Studio Code / Canva

ポストカード
「HAPPY HALLOWEEN」

ハロウィンのポストカードを制作する課題でした。ハロウィンらしい配色とフォントにすることを心掛けました。

「右側のかぼちゃと同じものを左側から見つけてもらう」という要素を入れて、少しでも長く目に留めてもらうようなデザインにしました。文字を囲っている黒い線は、太さを不均一にして蜘蛛の巣っぽくしてみました。

かぼちゃの後ろには・・・白いオバケがいます・・・。悪さをするとオバケに連れていかれてしまう・・・かも？

制作時期
2023年10月31日

使用ツール:Illustrator

Future map
「I WANT TO BE LIKE THIS IN THE FUTURE.」

職業訓練校の課題で自分のFuture mapを制作しました。

今の自分自身が黒背景で、未来にやりたいことや、こうなっていたいということをカラフルな流体シェイプにしました。

現状は将来やりたいことがあまり思いつかず、黒い背景が目立ちますが、今後はこのカラフルなシェイプ(＝夢)が増えて、この黒背景を埋め尽くしてくれるといいな…と願いを込めて作りました。

制作時期
2023年9〜10月頃

使用ツール:Photoshop
